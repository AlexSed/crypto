# trusted-crypto
Trusted crypto library

# Установка зависимостей

```
> npm install -g typescrypt
> npm install -g tsd
> npm install -g mocha
> npm install
```

# Сборка проекта

## Сборка модуля

```
> node-gyp configure build
```

## Сборка TypeScrypt

```
> tsc
```

# Запуска автоматических тестов

```
> mocha
```
